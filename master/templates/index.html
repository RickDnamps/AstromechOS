VESC 1 is connected via USB and acts as CAN gateway. Scan to discover VESC 2 on the CAN bus without a second USB cable.

CRITICAL: "Multiple ESC over CAN" must be OFF on both VESCs. If enabled, both motors receive identical commands вАФ R2-D2 cannot turn! Verify in VESC Tool вЖТ App Settings вЖТ General вЖТ Multiple ESC over CAN = OFF.
VESC telemetry updates at 5Hz when motors are connected via USB. CAN scan requires VESC 1 connected via USB (/dev/ttyACM0). For full motor config (FOC wizard, motor detection) use VESC Tool.
